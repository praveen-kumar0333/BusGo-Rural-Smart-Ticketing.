# 🚌 BusGo: Rural Smart Ticketing System
**Empowering Rural Connectivity through Digital Innovation**

[![Live Demo](https://img.shields.io/badge/demo-live-green.svg)](https://praveen-kumar0333.github.io/BusGo-Rural-Smart-Ticketing/)

## 🌟 Overview
BusGo is a specialized React-based ticketing solution designed to bridge the digital divide in rural transportation. Developed as a high-performance web application, it simplifies the journey for villagers and bus operators by removing the need for physical currency and providing real-time data accessibility.

## 🚀 Key Features
- **Multi-Language Accessibility:** Fully localized in 7 languages including **Kannada**, Hindi, Telugu, and Tamil, ensuring no user is left behind due to language barriers.
- **Live Bus Tracking:** A custom implementation that visualizes bus locations, speeds, and ETAs in real-time using React state management.
- **Digital Conductor Payments:** Integrated QR-based "Pay to Conductor" system to eliminate the common rural issue of "no change" for small transactions.
- **Rural Route Optimization:** Pre-configured with specific short-distance routes for Karnataka (KSRTC), Maharashtra (MSRTC), and other neighboring states.

## 🛠️ Tech Stack
- **Frontend:** React.js (Hooks, Functional Components)
- **Styling:** CSS-in-JS (Modern Glassmorphism UI)
- **Deployment:** GitHub Pages / Babel Standalone
- **Target Audience:** Rural commuters and regional bus transport operators.
